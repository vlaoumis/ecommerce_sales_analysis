# E-commerce Sales Analysis

This is a beginner-friendly data analysis project using Python, Pandas and Matplotlib.

## Project Goal

The goal of this project is to analyse a small e-commerce sales dataset, calculate revenue, identify top products, analyse customer behaviour and visualise monthly sales performance.

## Tools Used

- Python
- Pandas
- Matplotlib
- Google Colab

## Analysis Performed

- Data loading
- Basic data exploration
- Missing values check
- Revenue calculation
- Revenue by product
- Revenue by category
- Monthly revenue analysis
- Average order value
- Orders per customer
- CSV export

## Key Findings

- Electronics generated the highest revenue.
- Laptop and Phone were the strongest products by sales value.
- Some customers made repeated purchases.
- Monthly revenue increased across the sample period.

## Skills Demonstrated

- Python programming
- Pandas data analysis
- Data cleaning
- Grouping and aggregation
- Business metrics
- Data visualisation
